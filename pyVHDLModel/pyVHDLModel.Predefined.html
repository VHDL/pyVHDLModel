pyVHDLModel.Predefined

This module contains base-classes for predefined library and package declarations.

Classes

  • PredefinedLibrary: A base-class for predefined VHDL libraries.

  • PredefinedPackageMixin: A mixin-class for predefined VHDL packages and package bodies.

  • PredefinedPackage: A base-class for predefined VHDL packages.

  • PredefinedPackageBody: A base-class for predefined VHDL package bodies.
